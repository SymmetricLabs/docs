## Welcome to Tesseract Workflow API

The Tesseract Workflow API provides a comprehensive set of endpoints for managing and executing automated workflows. This RESTful API allows you to:

- **Start and stop workflows** by ID with custom parameters
- **Monitor workflow execution** in real-time
- **Retrieve workflow details** including input/output schemas
- **Manage workflow runs** with full CRUD operations

## Response Format

All API responses follow a consistent format:

```json
{
  "status": "success|error",
  "message": "Human readable message",
  "data": {
    // Response data object
  }
}
```
