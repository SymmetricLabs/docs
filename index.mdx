## Core Features

Discover what you can build with the Tessa Workflow API:

- **Workflow Management**: Create, start, and stop workflows programmatically
- **Health Monitoring**: Check API health status and monitor workflow execution states
- **Workflow Details**: Retrieve and manage workflow configurations and settings
- **Status Tracking**: Get real-time status updates for any workflow run
- **Run Management**: Manipulate workflow runs and manage execution history
